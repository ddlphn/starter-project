# Starter Kit

A simple Gulp Starter Kit for testing stuff.

## Installation

```sh
npm install
```

Start local server + watch files modifications
```sh
gulp 
```

Build all assets
```sh
gulp build
```

### Dependencies

@babel/core
@babel/preset-env
browser-sync
gulp
gulp-autoprefixer
gulp-babel
gulp-concat
gulp-rename
gulp-sass
gulp-uglify
jquery
sass
